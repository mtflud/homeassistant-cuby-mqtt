# Cuby (MQTT)

Home Assistant integration for [Cuby](https://cuby.mx) AC controllers, ported from the Homebridge plugin. Uses the built-in MQTT integration — no extra broker config needed.

Each Cuby device exposes a Climate entity, current temperature & humidity sensors, signal strength, and optional Display / Turbo / Long / Eco switches.

See the [README](https://github.com/mtflud/homeassistant-cuby-mqtt) for setup.
